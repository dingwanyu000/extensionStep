# extensionStep
a extension step of element-ui

基于Element-ui的扩展步骤条组件

API
-----------------------
|Param          | Description   | Type  |
| ------------- |:-------------:| -----:|
| size          | 每行显示个数   | Number|
| data          | 显示数据       | Array |

|event          | Description   | param |
| ------------- |:-------------:| -----:|
| listenToChild | 组件回掉事件   | data.id|
